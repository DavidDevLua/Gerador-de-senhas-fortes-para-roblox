1. Baixe o arquivo geradorsenha.py
2. Abra o Prompt de Comando (CMD) e navegue até a pasta onde salvou o arquivo:
cd C:\Users\SeuNome\Downloads
3. Execute o script:
py geradorsenha.py
4. Escolha o tamanho da senha digitando 1, 2 ou 3 e pressione Enter:
=== Gerador de Senhas para Roblox ===

Escolha o tamanho da senha:
1 - 15 caracteres
2 - 25 caracteres
3 - 45 caracteres

Digite 1, 2 ou 3:
5. A senha aparece na tela — é só copiar e usar!
⚠️ Aviso
Use senhas geradas apenas em suas próprias contas. Nunca compartilhe sua senha com ninguém, obrigado.
